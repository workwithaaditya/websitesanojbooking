# 📸 Image Guide — Sanoj Travels

## Folder Structure
```
images/
├── tours/        ← Tour destination photos
├── services/     ← Insurance & financial service images
├── icons/        ← Train, flight, bus, taxi logos/icons
├── hero/         ← Homepage banner images
```

## Naming Convention
Use **lowercase**, **no spaces** (use hyphens `-`), and keep names short.

## Image Tips
- **Size**: Keep under 500KB per image (compress at tinypng.com)
- **Dimensions**: Tours 800x500px, Icons 200x200px, Hero 1920x800px
- **Format**: `.jpg` for photos, `.png` for logos/icons with transparency
